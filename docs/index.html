Práctica 6 DSI: Clases e interfaces genéricas: Principios SOLID

Introducción

En esta práctica aprenderemos a utilizar las clases e interfaces genéricas, así como los principios SOLID. Para ello se nos pude realizar una serie de ejercicios en los que tenemos que implementar una serie de clases y interfaces genéricas.

Ejercicio 1: DSIflix

En este ejercicio vamos a crear una interfaz genérica que trata de especificar las propiedades que permitan por ejemplo obtener listados en función a difentes términos de búsqueda, también se nos pide crear una clase genérica BasicStreamableCollection que implemente la interfaz anteriormente creada. Por último, de dicha clase genérica se crean tres clases hijas, una para películas, otra para series y otra para documentales. La implementación sería la siguiente:

export interface Streamable<T> {
  buscarNombre(predicado: T): T[] | undefined;
  buscarAnio(predicado: number): T[] | undefined;
  buscarGenero(predicado: T): T[] | undefined;
}

export abstract class BasicStreamableCollection<T> implements Streamable<T> {
  constructor(protected collection: T[][]) {
  }
  /**
   * Metodo getCollection
   * @returns devuelve la colección
   */
  getCollection() {
    return this.collection;
  }
  abstract buscarNombre(predicado: T): T[] | undefined;
  abstract buscarAnio(predicado: number): T[] | undefined;
  abstract buscarGenero(predicado: T): T[] | undefined;
  /**
   * Metodo insertar
   * @param item elemento a insertar
   */
  insertar(item: T[]): void {
    this.collection.push(item);
  }
}

export class Peliculas<T> extends BasicStreamableCollection<T> {
  constructor (collection: T[][]) {
    super(collection);
  }
  /**
   * Método que busca una pelicula por su nombre
   * @param predicado nombre de la pelicula
   * @returns lista de peliculas con el nombre indicado
   */
  buscarNombre(predicado: T) :T[] | undefined{
    let resultado: T[] = [];
    for (let i = 0; i < this.getCollection().length; i++) {
      if (this.getCollection()[i][0] === predicado) {
        resultado.push(this.getCollection()[i][0]);
      }
    }
    return resultado;
  }
  /**
   * Método que busca una pelicula por su año
   * @param predicado año de la pelicula
   * @returns lista de peliculas con el año indicado
   */
  buscarAnio(predicado: number) :T[] | undefined{
    let resultado: T[] = [];
    for (let i = 0; i < this.getCollection().length; i++) {
      if (this.getCollection()[i][1] === predicado) {
        resultado.push(this.getCollection()[i][0]);
      }
    }
    return resultado;
  }
  /**
   * Metodo que busca una pelicula por su genero
   * @param predicado genero de la pelicula
   * @returns lista de peliculas con el genero indicado
   */
  buscarGenero(predicado: T) :T[] | undefined{
    let resultado: T[] = [];
    for (let i = 0; i < this.getCollection().length; i++) {
      if (this.getCollection()[i][2] === predicado) {
        resultado.push(this.getCollection()[i][0]);
      }
    }
    return resultado;
  }
}

En esta implementación, la clase BasicStreamableCollection es la clase abstracta que implementa la interfaz Streamable y que contiene los métodos comunes a las clases Peliculas, Series y Documentales. Estas tres clases heredan de BasicStreamableCollection y sobreescriben los métodos buscarNombre, buscarAnio y buscarGenero para que funcionen de forma diferente en cada una de ellas. Estos métodos nos permiten buscar una película, serie o documental por su nombre, año o género. Para ello, recorremos la colección de películas, series o documentales y comparamos el nombre, año o género de cada elemento con el predicado que le pasamos como parámetro. Si coinciden, añadimos el elemento a la lista de resultados y la devolvemos.

Ejercicio 2: implementación de una lista y sus operaciones

En este ejercicio se nos pide crear una clase genérica que modele una lista de elementos y que define ciertas operaciones como append, concatenate, filter, length, map, reduce, reverse y forEach. Pero para ello no se nos permite utilizar las funcionalidades proporcionadas por Array.Prototype. En su lugar, debemos implementar las operaciones de forma manual. La implementación de la clase es la siguiente:

export abstract class ListaElementos<T> {
  constructor (protected collection: T[]) {
  }
  /**
   * Método getCollection
   * @returns devuelve la colección
   */
  getCollection() {
    return this.collection;
  }
  /**
   * Método length
   * @returns devuelve el tamaño de la colección
  */
  length(): number {
    let count = 0;
    while (this.collection[count] !== undefined) {
      count++;
    }
    return count;
  }
  /**
   * Método append
   * @param item lista de elementos a añadir
   * @returns lista de elementos con los elementos añadidos
   */
  append(item: Lista1<T>): T[] {
    let tamano = this.collection.length;
    for(let i = 0; i < item.getCollection().length; i++) {
      this.collection[tamano + i] = item.getCollection()[i];
    }
    return this.collection;
  }
  /**
   * Método concat
   * @param item lista de elementos a concatenar
   * @returns lista de elementos con los elementos concatenados
   */
  concat(...item: Lista1<T>[]): T[] {
    let resultado = new Lista1<T>(this.collection);
    for(let i = 0; i < item.length; i++) {
      const element = item[i];
      resultado.append(element);
    } 
    return resultado.getCollection();
  }
  /**
   * Método filter
   * @param predicado predicado a aplicar
   * @returns lista de elementos filtrados
   */
  filter(predicado: (item: T) => boolean): T[] {
    let resultado: T[] = [];
    let indice = 0;
    for (let i = 0; i < this.collection.length; i++) {
      if (predicado(this.collection[i])) {
        resultado[indice] = this.collection[i];
        indice++;
      }
    }
    return resultado;
  }
  /**
   * Método map
   * @param predicado predicado a aplicar
   * @returns lista de elementos con el predicado aplicado
   */
  map<U>(predicado: (item: T) => U): U[] {
    let resultado = new Lista1<U>([]);
    for (let i = 0; i < this.collection.length; i++) {
      let elemento = this.collection[i];
      resultado.append(new Lista1<U> ([predicado(elemento)]));
    }
    return resultado.getCollection();
  }
  /**
   * Método reduce
   * @param predicado predicado a aplicar
   * @param valorInicial valor inicial
   * @returns valor final
   */
  reduce<U>(predicado: (item: T, item2: U) => U, valorInicial: U): U {
    let resultado = valorInicial;
    for (let i = 0; i < this.collection.length; i++) {
      resultado = predicado(this.collection[i], resultado);
    }
    return resultado;
  }
  /**
   * Método reverse
   * @returns lista de elementos invertida
   */
  reverse (): T[] {
    let resultado = new Lista1<T>([]);
    for (let i = this.collection.length - 1; i >= 0; i--) {
      resultado.append(new Lista1<T> ([this.collection[i]]));
    }
    return resultado.getCollection();
  }
  /**
   * Metodo forEach
   * @param predicado predicado a aplicar
   */
  forEach (predicado: (item: T) => void): void {
    for (let i = 0; i < this.collection.length; i++) {
      predicado(this.collection[i]);
    }
  }
}

En esta implementación he creado una clase genérica en la cuál se almacena una lista de elementos y se definen las operaciones que se pueden realizar sobre ella. Para ello, he creado un constructor que recibe como parámetro la lista de elementos y un método getCollection que devuelve la lista de elementos. Además, he creado un método length que devuelve el tamaño de la lista de elementos. Para implementar el método append, he creado una variable que almacena el tamaño de la lista de elementos y he recorrido la lista de elementos que se le pasa como parámetro y he añadido cada elemento a la lista de elementos de la clase. Para implementar el método concat, he creado una variable que almacena la lista de elementos de la clase y he recorrido la lista de elementos que se le pasa como parámetro y he añadido cada elemento a la lista de elementos de la variable. Para implementar el método filter, he creado una lista vacía y he recorrido la lista de elementos de la clase y he añadido a la lista vacía los elementos que cumplan el predicado. Para implementar el método map, he creado una lista vacía y he recorrido la lista de elementos de la clase y he añadido a la lista vacía los elementos que cumplan el predicado. Para implementar el método reduce, he creado una variable que almacena el valor inicial y he recorrido la lista de elementos de la clase y he aplicado el predicado a cada elemento y he almacenado el resultado en la variable. Para implementar el método reverse, he creado una lista vacía y he recorrido la lista de elementos de la clase de forma inversa y he añadido a la lista vacía los elementos. Para implementar el método forEach, he recorrido la lista de elementos de la clase y he aplicado el predicado a cada elemento.

export class NumericPrintableCollection extends PrintableCollection<number>{
  constructor (protected collection : number[]) {
    super(collection);
  }
  /**
   * Método print para imprimir
   * @returns string con los datos de la colección separado por comas
   */
  print () : string {
    return this.collection.join(',');
  }
}
